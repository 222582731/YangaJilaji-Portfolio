Tutor Finder System

A full-stack web application designed to connect students with suitable tutors for academic support. The platform enables secure authentication, tutor searching, and direct communication between students and tutors. Students can browse tutor profiles, search by subject, and leave ratings and feedback. Built using Java Spring Boot, RESTful APIs, MySQL, and a responsive frontend with HTML, CSS, and JavaScript.

School Management System

A full-stack web application designed to streamline academic management and enhance communication between students and teachers. The system supports secure login and role-based access, allowing students to track attendance, view grades, download coursework, and communicate with teachers. Teachers can upload coursework, manage attendance and grades, and send messages. Built using Java Spring Boot, RESTful APIs, MySQL, and a responsive frontend with HTML, CSS, and JavaScript.

Frontend Backend

Tymelesstyre E-Commerce

A full-stack e-commerce platform for selling car tyres and rims online. Customers can browse products, add items to their cart, and place orders securely. The system includes JWT-based authentication and role-based access control. An admin dashboard allows administrators to add, update, and remove products while managing inventory. Built with Java Spring Boot, Spring Security, REST APIs, and a Vue.js frontend.
